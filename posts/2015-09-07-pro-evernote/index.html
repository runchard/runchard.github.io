我认为电子笔记最重要的两个功能:

  1. 累积(方便记录, 整理各类信息)
  2. 回忆(方便查找, 反馈各类信息) 其他所有的功能都是为这两种功能提供服务. 本文所有技巧都是通过改善累积笔记的方式和习惯来提高累积的效率和回忆的能力.

Evernote 产品本身 Evernote 对单则笔记的漂亮程度其实并不要求.

笔记的组织形式也很简单, 就是bookname+notes. 但是有意义的是组织笔记相关间关系的方式有灵活的tag 和 inner hyper link. 对笔记理解更加深入之后, 感觉这才是正确的记录笔记的方式.

例如本文的采用的符号「」, 都是典型的简单但是能够促进阅读的东西. 但是一旦查找最近的笔记会感觉很苦恼, 那就需要花费一定时间去安排组织笔记了.

Thread is everywhere - 保持记忆线索保持线索可以厘清思路, 也可以帮助你很快的回想起相关的内容.

Meaningful Order - 有意义的顺序Evernote 的记事本会自动排序, 但是可以通过数字编号来固定排序方式.

把最重要的笔记本冠以「0」, 「1」这样的数字, 然后通过名称规则排列在前. 这也是高效能人士所推崇的存储联系人的方式.

Copy Note Link这是我认为特别重要的一个功能, 通过他, 你可以轻易的处理笔记间的链接.

在复杂的逻辑和下面的「学习笔记」中, 你会经常用到它.

how to use note links

Power TagTag是印象笔记中特别重要的功能. 他可以让你解析梳理各种笔记之间关系.

你可以轻易的在工作分类和生活分类中加入权重tag 例如 「!」, 「#」, 这样就可以很快的排序和查找到自己需要的内容.

也可以加入GTD相关的tag, 这样GTD相关的信息可以有条理的分散在各个笔记本当中.

clear your mind - 明确意图尽量使用统一的格式去梳理笔记中各类标记的意图

比如

  • 贯穿效果就意味着不要使用或者不明这段文字的意图;
  • 高亮效果就意味着这里有问题需要特别注意;

  • 斜线表示插入和引用.

Checkbox 非常重要!

一个笔记里面的待办都是应该以checkbox的形式去展现. 一旦习惯了这种规律, 那么所有的细节都将会很清晰. 这些简单的标记在逻辑清晰的情况下, 足够95%的使用案例.

Clean your data重点突出, 解释详细

好处有三:

  • 让 Note 尽可能稳定, 不会尝试触碰 evernote 的崩溃界限. 丢了笔记的代价是很昂贵的;
  • 使用简化格式使得你尽可能的注重思考内容和笔记本身;
  • 减少在美化表面上浪费的时间, 艺术本身是无底洞.
  • 原则是只要使用「简化格式」(simplify formatting) 这个操作就会丢失的任何格式都不要使用.

精简 Notebooks

一旦当你开始决定计划认真使用evernote的时候, 很容易做的一件事情就是, 添加各种notebook, 但是, 其实你没那么多分类. 先想想事情, 然后直接添加一则笔记即可.

建立新笔记本的原则

  • 这个笔记本我一定会用到
  • 让我看到分类时，马上就知道怎么依据分类来行动。
  • 定义清晰稍后阅读层次, 养成良好的阅读习惯和整理习惯.

Keep deleting

我反复的看了我三年前的笔记, 幼稚的令人发至, 我不想看到他, 但是有不舍得删去他

一旦完成一则笔记, 封存或者删除. 当然, 这意味着你首先需要建立一个 archive 的 notebook. 你不需要资料库, 资料库很容易过期, 你只需要你自己的知识库, 让 Google 或者其他东西成为你的资料库.

删除就意味着这个笔记过期或者不值得继续记录.

Note Cases - 笔记的典型案例

Learning Notes

学习笔记越难的目标越要从基础做起, 学习一项复杂的事物, 比如写论文, 那么你会遇到很多的细节需要处理. 你可能会发现亮点, 但是你需要组合这一切, 你需要checkbox来标记未完成的事项.

Reading Notes - 读书笔记笔记是一部分, 整理和思考又是一部分, 借鉴之前的tag和索引就会帮助你厘清思路.

Diary, jottings - 日记,

随感

这些内容无论以哪种方式呈现, 分类都可能出现问题. 我曾反复的删除添加过我的日记和对生活的感悟这种笔记本. 我现在会把它直接放到生活中去, 不去考虑太多. 搜索会帮助我找到我需要的内容.

Relationships - 处理人脉关系

利用 Evernote 处理人脉管理, 而非简单的通讯录. 要考虑和别人合影或者拍一张. 之后发给对方, 方便提醒自己这个人是谁, 也方便进行之后交流的破冰. 之后即时建立 checkbox 去整理个人信息, 通过 evernote Contacts OCR. 之后记录相关信息的时候, 可以直接利用Contact的「Copy Note Link」就很清楚自己做了什么了.

Using Search 利用搜索

「看书得知 evernote 早年是做OCR起家的」

如果只是为了查找某些笔记, 利用搜索往往比你自己整理好的路径还快.

搜索可以使用空格来区分不同的关键字, 提升被搜索的概率. 另外 evernote 虽然不能主动提取出可供选择的文字, 但是支持搜索. 这样模糊搜索应该查找到需要内容的概率会更高.

zhihu.com 就是一个内容整理失败的无底洞, 你看起来能收藏, 其实只能依靠类似tag的系统去收藏, 一旦你回忆起某一个答案的部分细节, 那么你一定再也搜索不到你需要的内容了.

Be Careful about the extensionsEvernote 的各项功能已经足够使用.

虽然看起来有很多很好用的第三方 app 存在, 但是如果我们不小心的使用各类拓展就会造成可能的数据丢失和泄漏. Evernote 自身的权限控制并不是非常合理. 所以并不推荐使用 evernote 非官方的各类拓展.
